# TeamFinder — Вариант 2 (Система поиска команд по навыкам)

Платформа для поиска единомышленников и работы над pet-проектами.

## Стек технологий и архитектура
* **Язык разработки:** Python 3.11
* **Фреймворк:** Django 5.2
* **База данных:** PostgreSQL 15 (в Docker-окружении) / СУБД SQLite (для локального тестирования)
* **Контейнеризация:** Docker, Docker Compose
* **Конфигурация окружения:** Python-decouple (управление секретами через `.env`)
* **Форматирование кода:** Black (соответствие стандарту PEP 8)

---

## Инструкция по развёртыванию и запуску

Архитектура проекта поддерживает автоматическое переключение конфигураций: при запуске на локальной машине используется встроенная СУБД SQLite для изоляции процесса разработки, а при развёртывании через Docker Compose автоматически поднимается промышленный стек с PostgreSQL.

### 1. Настройка переменных окружения
В корневой директории проекта (на одном уровне с файлом `manage.py`) создайте файл `.env` и добавьте в него следующие конфигурационные ключи:

```env
DJANGO_SECRET_KEY=temporary-secret-key-for-review-123
DJANGO_DEBUG=True
POSTGRES_DB=postgres
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres
POSTGRES_HOST=db
POSTGRES_PORT=5432
RUNNING_IN_DOCKER=True
